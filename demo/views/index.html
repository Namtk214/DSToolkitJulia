🚀 DSToolkit Demo

Machine Learning Toolkit for Julia

Features

  • 9 Sample Datasets (Classification, Regression, Time Series)
  • 22 ML Algorithms
  • Data Preprocessing Pipeline
  • Auto-Compare Models
  • Visualization
  • Model Persistence

Quick Start

using DSToolkit

# Load dataset
data = load_iris()

# Train and compare models
result = auto_compare(data)

# Make predictions
predictions = predict(result.best_model, X_test)
